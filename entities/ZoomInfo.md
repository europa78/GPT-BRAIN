# ZoomInfo
> One-liner: Mike's primary B2B contact-data platform — ICP list building and email-template targeting for both agency prospecting and investor outreach; the standard upload target for scraped lead lists.

## How it's used in practice
- **ICP lists + email templates** for [[STB Digital]] (2025-07-22, "ICP client list for ZOOMinfo" adjacent).
- **Agent-mode integration:** the lead-gen agent blueprint wires **ZoomInfo → [[SalesLoft]] → Gmail** as connectors (2025-08-02).
- **Upload target for scrapes:** Sunbiz/Broward new-LLC scrape → CSV → "upload the csv to ZoomInfo" (2025-08-03).
- **Qualification workflows** (2025-09-07, "zoominfo qualification glen gary").
- Same platform serves [[mPWR]]/[[ALBT]] investor list building.
