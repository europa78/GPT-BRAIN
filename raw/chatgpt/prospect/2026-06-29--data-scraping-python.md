```python
import openpyxl

# Make an HTTP request to the website
url = 'https://www.example.com/small-business'
response = requests.get(url)

# Parse the HTML content using BeautifulSoup
soup = BeautifulSoup(response.content, 'html.parser')
table = soup.find('table')

# Create a new workbook and worksheet
wb = openpyxl.Workbook()
ws = wb.active

# Write the table headers to the worksheet
headers = [th.text for th in table.find_all('th')]
ws.append(headers)

# Write the table data to the worksheet
for tr in table.find_all('tr')[1:]:
    row = [td.text for td in tr.find_all('td')]
    ws.append(row)

# Save the workbook to a file
wb.save('small_business.xlsx')
```

In this example, we are copying a table from a website and importing it into an Excel worksheet. We use the requests library to make an HTTP request to the website's URL, and then use BeautifulSoup to parse the HTML content and extract the table data. We then create a new workbook and worksheet using openpyxl, and loop through the table rows to write the data to the worksheet using the append() method. Finally, we save the workbook to a file.

Note that this is just a simple example, and the specific details of how to extract data from a website will depend on the structure of the website and the format of the data that you want to copy. Additionally, you may need to perform additional data manipulation or formatting operations to get the data in the format that you need for your Excel worksheet.
