**GPT:** Sure, here's a simple Python script using the PyTrends library to get Google Trends data for three keywords on a daily basis. Please replace 'keyword1', 'keyword2', and 'keyword3' with your actual keywords.

```python
from pytrends.request import TrendReq
import time
import schedule

# Set up pytrends
pytrends = TrendReq(hl='en-US', tz=360)

# Define the keywords
keywords = ['keyword1', 'keyword2', 'keyword3']

def get_trends():
    # Build the payload
    pytrends.build_payload(keywords, cat=0, timeframe='now 1-d', geo='', gprop='')

    # Get Google Trends data
    data = pytrends.interest_over_time()

    # Print the data
    print(data)

# Schedule the job every day
schedule.every().day.at("00:00").do(get_trends)

# Keep the script running
while True:
    schedule.run_pending()
    time.sleep(1)
```

This script will run the `get_trends` function every day at midnight. The function gets the Google Trends data for the specified keywords over the past day and prints the data.

Please note that this is a basic script and you might want to add error handling and data storage capabilities depending on your needs. Also, as mentioned before, PyTrends is an unofficial API for Google Trends, so use it responsibly to respect Google's Terms of Service.
