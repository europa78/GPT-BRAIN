## **1. Overview**  
Stable Audio allows users to generate AI-based audio with various inputs, including text prompts, uploaded audio files, and recorded clips. This guide explains how to use these features effectively.

---

## **2. Audio-to-Audio Feature**  
The audio-to-audio feature enables users to modify and enhance existing audio by applying AI transformations. Users can:  

- Upload a file  
- Pair it with a text prompt  
- Adjust parameters such as input audio strength  

This allows for unique and dynamic audio transformations.

---

## **3. Types of Input Audio**  
Users can add different types of audio to guide AI-generated outputs. Options include:  

- **Generate Audio**: Use previously created Stable Audio files.  
- **Upload Audio**: Add external audio files (stems, samples, or full songs).  
- **Record Audio**: Instantly capture new audio using the built-in recorder.  
- **Vocal Input**: Hum or sing to create guided output.  

Adding input audio alongside a text prompt helps experiment with creative outputs.

---

## **5. Uploading & Recording Audio**  
Users can upload audio in various formats:  

- **Accepted Formats**: MP3, WAV, MP4, AIFF  
- **Max Length**: 3 minutes per upload  
- **Min Length**: 1 second  

Uploaded audio is stored separately and will **not** be used to train AI models.  

For best results, use **wired microphones** for recording, as Bluetooth mics introduce latency.

---

## **6. Monthly Upload Allowance**  
Stable Audio enforces a **monthly upload limit** based on the user's subscription tier:

| Plan  | Max Upload Time | Cropped Upload Length |
|--------|--------------|--------------------|
| Free   | 3 min/month  | 30 sec per file   |
| Pro    | 30 min/month | 3 min per file    |
| Studio | 60 min/month | 3 min per file    |
| Max    | 90 min/month | 3 min per file    |

If an uploaded file **violates copyright**, it still counts toward your upload limit.

---

## **7. Upload Details & Limitations**  
- Users must **own or have rights** to any uploaded audio.  
- Copyrighted files will be flagged but still count against the **monthly allowance**.  
- Users can track their monthly usage within the account settings.  
- Check Stable Audio’s **pricing page** for details on increasing upload limits.

## **8. Best Practices for Recording**  
To get the **best audio quality**, follow these tips:  

✅ **Use a wired microphone** (Bluetooth mics introduce 0.5s latency).  
✅ **Record externally** using professional equipment before uploading.  
✅ **Minimize background noise** for clean recordings.  
✅ **Experiment with different input strengths** to fine-tune results.  

By following these guidelines, you can achieve high-quality AI-generated audio.
