## The 6 “low-infrastructure” business models that still print money
1. **Paid license (desktop / local web app):** You ship a downloadable app that runs locally. Updates are optional paid upgrades.
2. **Embeddable widget:** A small script customers paste into their site (quote calculator, lead qualifier, scheduling pre-check). Can be “no backend” or minimal.
3. **Templates + generators:** A guided builder that outputs ready-to-use assets (landing pages, ads, emails, proposals) as files.
4. **BYOK AI tools:** The app calls OpenAI/etc using the customer’s API key → you don’t pay inference or host models.
5. **Marketplace piggyback:** Sell through existing rails (Shopify app store, Chrome store, WordPress plugins, Gumroad, Envato) so distribution + billing is handled.
6. **Productized “micro-enterprise” tool:** One narrow painful workflow, solved completely. Charge more because it replaces labor.
