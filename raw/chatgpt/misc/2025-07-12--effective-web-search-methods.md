### Quick-reference cheat sheet (print & stick on monitor)  
```
“exact phrase”   | match precise wording  
keyword -minus   | exclude a term  
OR / (x OR y)    | search variants  
site:domain.tld  | stay inside one domain  
filetype:pdf     | only PDFs  
after:YYYY-MM-DD | newest content  
!bang            | jump to another site (DDG)  
```
Use these seven commands with clear intent and AI cross-checks, and you’ll out-search 95 % of the web—while wasting a fraction of the time.

### 1. Boolean & basic syntax  
| Operator | Purpose | Quick example |
| --- | --- | --- |
| `"exact phrase"` | Match words in that order | `"digital marketing strategist"` |
| `-term` | Exclude a word/phrase | jaguars -speedway -NFL |
| `OR` / `\|` | Either term | stem cell OR “regenerative medicine” |
| `*` | Wildcard placeholder | “* is thicker than water” |
| `(..)` | Numeric range | 12..24 gb ram laptop |
| `( )` | Group logic | (marketing OR advertising) “stem cells” |
| `AROUND(N)` | Two terms within N words | wimbledon AROUND(3) upset citeturn0search3 |

### 2. Domain & URL filters  
| Operator | What it limits | Example |
| --- | --- | --- |
| `site:` | Whole domain / sub-folder | site:nytimes.com stem cells |
| `inurl:` | URL contains term | inurl:/careers “remote” |
| `allinurl:` | URL contains **all** terms after operator | allinurl:marketing proposal pdf |
| `intitle:` | Page title contains term | intitle:“Wimbledon preview” |
| `allintitle:` | Title contains **all** terms | allintitle:seo audit checklist |
| `intext:` | Body text contains term | intext:“SB 1768” |
| `allintext:` | Body text contains **all** terms | allintext:orthopedics stem cell florida |
| `inanchor:` / `allinanchor:` | Anchor text in backlinks (spotty support) | inanchor:“download white-paper” citeturn0search0 |

### 7. Combining operators—two power patterns  

1. **Competitive content gap**  
```bash
site:competitor.com -inurl:blog "stem cell therapy"
```
→ Lists every competitor page *except* blog posts that mention stem-cell therapy.  

2. **Fresh PDFs on a regulation**  
```bash
"SB 1768" filetype:pdf after:2025-07-01 site:.gov
```
→ Finds official government PDFs published since the law changed. citeturn0search1

## Tips for reliability  
1. Place *allinurl*, *allintitle*, etc. **after** other operators—Google ignores anything that follows them.  
2. Put no space after a colon (`site:` not `site :`).  
3. Layer filters sparingly; too many may return zero results even though pages exist.  
4. Remember date operators apply to when Google last indexed a page, **not** publication date. citeturn0search0
