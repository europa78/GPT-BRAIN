# 🔥 1. MASTER LANDING PAGE GENERATION PROMPT (FULL SITE)

Use this when you want a complete, polished, interactive one-page site.

```
ROLE:
You are a world-class UI/UX designer, conversion copywriter, and senior front-end engineer specializing in high-converting landing pages.

DIRECTIVE:
Create a production-quality, single-file landing page using HTML, CSS, and JavaScript.

CONTEXT:
Business Name: [INSERT]
Offer: [INSERT PRODUCT/SERVICE]
Target Audience: [INSERT ICP]
Primary Goal: [Leads / Sales / Book Calls / Signups]
Tone: [Luxury / Aggressive / Friendly / Minimalist / Corporate]
Style Inspiration: [Apple / Stripe / Tesla / SaaS / Dark Mode / Futuristic]

FEATURE REQUIREMENTS:
- Fully responsive (mobile-first)
- Sticky navigation bar
- Hero section with strong headline + CTA
- Dynamic elements (animations, hover effects, transitions)
- Interactive components (dropdowns, toggles, tabs, sliders, etc.)
- Lead capture form with validation
- Sections:
  - Hero
  - Social Proof (testimonials/logos)
  - Features/Benefits
  - How It Works
  - FAQ (collapsible)
  - Call-to-Action sections throughout
  - Footer

DESIGN REQUIREMENTS:
- Modern UI with clean spacing and hierarchy
- Smooth animations (CSS or JS)
- Visually engaging (gradients, shadows, depth)
- Consistent color palette and typography
- Conversion-focused layout (clear CTA emphasis)

COPY REQUIREMENTS:
- Write persuasive, benefit-driven copy
- Strong headline + subheadline
- Clear CTAs
- Objection-handling in FAQ
- No fluff — concise and impactful

TECHNICAL REQUIREMENTS:
- Deliver as a SINGLE HTML file
- Include embedded CSS and JavaScript
- Code must be clean, well-structured, and commented
- No external dependencies unless necessary

OUTPUT FORMAT:
Return ONLY the final HTML file code.
```
